ditherzam 0.1.0 for Windows x64 (portable)
================================================

1. Extract the entire ZIP to a folder you can write to.
2. Open that folder and double-click ditherzam.exe.

Keep the _internal folder beside ditherzam.exe. FFmpeg and FFprobe are bundled,
so video features do not require a separate install.

Smart Mask is disabled because this public build contains no model weights.

This build is unsigned. Windows SmartScreen may show a warning because the
publisher has no code-signing certificate. Do not disable SmartScreen; verify
the SHA-256 checksum published with the release.
